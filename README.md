# SQL-Superstore-Sales-Analysis

## Project Overview

This project focuses on analyzing a retail sales dataset using MySQL. The objective was to practice SQL while generating meaningful business insights from sales data.

## Tools Used

* MySQL Workbench
* SQL
* Superstore Sales Dataset

## Key SQL Concepts Applied

* SELECT
* WHERE
* DISTINCT
* COUNT()
* SUM()
* AVG()
* MIN() and MAX()
* GROUP BY
* ORDER BY
* LIMIT and OFFSET
* CASE WHEN
* HAVING

## Business Questions Answered

* Total records in the dataset
* Total sales by category
* Total sales by region
* Total sales by segment
* Top 10 customers by sales
* Top 10 products by sales
* Top 5 states by sales
* Bottom 5 states by sales
* Highest selling category
* Highest selling region
* Customer sales segmentation
* Categories with sales greater than a specified threshold

## Outcomes

This project helped strengthen SQL fundamentals and develop the ability to answer business questions using data analysis techniques.
